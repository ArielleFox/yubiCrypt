# yubiCrypt

Your Paranoid Encryption Tingi

# INSTALLATION

Running the installer script will install and set up interactivly with you...
Created folders:  ~/.yubiCrypt and ~/dcde
## Command
```bash
$ ./installer.py
```

# USAGE:
## File Encryption
```bash
$ yubienCrypt <filename>
```
## File Decryption
```bash
$ yubideCrypt <filename>
```
